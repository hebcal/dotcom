== Description ==

Displays Shabbat candle lighting times for a USA zip code in a
widget. Candle lighting time information comes from Hebcal.com.

Hebcal.com is a free Jewish calendar and holiday web site. It includes
a powerful custom Jewish calendar form that lets you generate a list
of Jewish holidays for any year (past, present or future). Also
available is a Hebrew date converter, Shabbat candle lighting times
and Torah readings, and a page to look up yahrzeits, birthdays and
anniversaries.
